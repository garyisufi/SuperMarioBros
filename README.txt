Due to the large size of the wildcats_sprint1 folder, here are the instructions on how to run the game:

1. Download the wildcats_sprint1 zip folder to your computer.
2. Extract the zip folder.
3. Enter the extracted zip folder.
4. Double click on the executable file located in wildcats_sprint1/bin/Windows/x86/Debug
	(the executable file is named: wildcats_sprint1)
5. Allow the game to run if a pop up occurs.
6. Raise volume and enjoy!
